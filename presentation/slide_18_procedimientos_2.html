Procedimientos Operativos II

Asignación de Sucesor
Pasos para Asignar Sucesor
1 Acceder a puesto crítico existente en el sistema
2 Seleccionar opción "Agregar Sucesor"
3 Buscar colaborador en base de datos de empleados
4 Seleccionar colaborador de la lista de resultados
5 Sistema auto-completa: cargo actual y departamento
6 Guardar registro y confirmar asignación
Validaciones de Integridad de Datos
✓ Existencia de Sucesor: Sucesor debe existir en tabla de empleados activos
✓ No Duplicación Directa: No puede ser el mismo colaborador del puesto crítico
✓ Alerta de Duplicación: Alerta si sucesor se asigna a múltiples puestos (advertencia amarilla)
✓ Auto-completado: Cargo y departamento se completan automáticamente desde tabla empleados
✓ Integridad Referencial: Se verifica relación válida entre tablas para evitar registros huérfanos
Responsables
Gerencia RH + Líderes Departamento
Frecuencia
Mensual (según cambios)
