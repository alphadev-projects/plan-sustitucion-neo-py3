Seguridad y Confidencialidad

Niveles de Acceso a Datos
Nivel Descripción Datos Accesibles
Público Información general del proceso Descripción del sistema, políticas, objetivos
Confidencial Datos operacionales del sistema Puestos críticos, sucesores, planes de acción
Muy Confidencial Información sensible y estratégica Evaluaciones, salarios, planes de carrera, decisiones estratégicas
Protecciones de Datos
✓ Cifrado en tránsito: Todos los datos se transmiten mediante protocolo HTTPS/TLS
✓ Cifrado en reposo: Base de datos cifrada con estándar AES-256
✓ Enmascaramiento de datos: Datos sensibles enmascarados en reportes no autorizados
✓ Auditoría de acceso: Registro de quién accede qué datos y cuándo
Medidas de Seguridad Operacional
• Autenticación multifactor: Requiere usuario, contraseña y código de verificación
• Control de sesión: Timeout automático después de 10 minutos de inactividad
• Respaldos automáticos: Copias de seguridad diarias con replicación geográfica
• Monitoreo continuo: Detección de anomalías y patrones de acceso sospechosos
• Cumplimiento normativo: Adherencia a GDPR, LGPD y regulaciones locales de protección de datos
